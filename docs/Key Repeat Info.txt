PET/CBM Editor ROM Project - PET "Key Repeat" Info
==========================   KEY REPEAT INFO.TXT

* VICE: Both Symbolic and Positional use pc's PGDN key as REPEAT

* Only the "B" keyboard has a repeat key
* In B Key scanner it will autorepeat the following keys:
	DEL, SPACE, CURSOR KEYS
* the Repeat Flag is: RPTFLG = $E4 - $80=Enabled, $40=Disabled
* CODEBASE 1 Usage:
	$E02D - Editor ROM Jump Table. Calls $E036 (this is not called within Editor ROM)
	$E036 - Set Repeat Flag using .A
	Keyboard Scanner
